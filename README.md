
#  VPS开启root用户登录

###  修改SSH端口和root密码

###  一键脚本：

```
wget -N --no-check-certificate https://github.com/taotao1058/rootvps/raw/main/rootvps && bash rootvps
```

---

```
sudo wget -N --no-check-certificate https://github.com/taotao1058/rootvps/raw/main/xin && sudo bash xin
```
